# Contributing

欢迎为 OpenQuality-Datasets 补充与“大质量”主题相关的公开数据集。

## 可收录的数据集方向

- 工业质量
- 制造质量
- 缺陷检测
- 设备健康监测
- 故障诊断
- 可靠性分析
- 半导体良率分析
- 数据质量
- 软件质量
- AI 质量

## 提交要求

新增数据集时，请尽量包含以下信息：

- 数据集名称
- 数据类型
- 所属领域
- 质量主题
- 适合任务
- 简要介绍
- 公开链接
- 许可证或使用限制
- 推荐引用方式

## 注意事项

本项目不直接重新分发原始数据文件，仅整理公开数据集的信息、链接和使用建议。请遵守原始数据集的许可证、引用要求和使用限制。
